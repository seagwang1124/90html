1) 성경을 읽기 전, 기도

페이지에 접속하면 우하단에 “기도하고 읽기” 팝업이 나타납니다. 안내에 따라 짧게 기도한 뒤 확인을 누르세요.

팝업은 다른 모달이 열려 있으면 자동으로 숨겨집니다. 닫더라도 페이지를 새로 열면 다시 표시됩니다.

2) 검색어 기능

오디오를 들으며 본문을 읽다가 위치를 잃었을 때, 오디오에서 들린 단어를 상단의 검색창에 입력하세요.

  • Enter 또는 검색 버튼으로 검색합니다.
  • 본문의 일치 단어가 노란색으로 강조됩니다.
  • 여러 곳이 일치할 경우, 화면 오른쪽에 빨간 점들이 나타납니다. 점을 탭/클릭하여 해당 위치로 즉시 이동할 수 있습니다.
  • 하단의 다음 위치 버튼으로 다음 검색 결과로 이동합니다.
  • X 버튼을 누르면 검색 하이라이트가 초기화됩니다.

모바일에서 검색창과 시간을 겹치지 않게 하기 위해 검색창은 화면 크기에 맞추어 자동으로 줄어듭니다.

3) 폰트 크기 조절

본문 위에 있는 Font Size의 +, − 버튼으로 글자 크기를 조절합니다.

  • 최소/최대 범위를 넘어가지 않도록 안전하게 제한되어 있습니다.
  • 가독성을 위한 중앙 두 줄 강조 기능과 함께 쓰면 장시간 읽기에도 편합니다.
Font Size 20

4) 오디오 컨트롤

상단 오른쪽의 오디오 컨트롤로 재생/일시정지/정지, 재생 위치, 재생 속도, 볼륨을 조절할 수 있습니다.

4-1. 기본 조작

▶ 재생
오디오를 시작합니다.
⏸ 일시정지
현재 위치에서 멈춥니다.
■ 정지
재생을 멈추고 처음 위치로 돌아갑니다.

4-2. 재생 위치

오디오 컨트롤 아래의 “재생 위치” 버튼을 누르면 막대(slider)와 10초 이동 버튼이 나타납니다.

  • 막대를 드래그해 원하는 시점으로 이동
  • ⏪10s: 10초 뒤로 · 10s⏩: 10초 앞으로

4-3. 재생 속도

“재생 속도” 버튼으로 0.5× ~ 2.0× 범위에서 0.1~0.2 단위로 조정하거나, 1.0 버튼으로 즉시 초기화합니다.

4-4. 볼륨 조절

“볼륨 조절” 버튼으로 볼륨 막대와 음소거 버튼(🔊/🔇)을 표시하여 조정합니다.

모바일에서 시간 잘림이 보일 때
남은 시간(-0:00)이 오른쪽에서 잘리면 화면이 아주 좁거나 시스템 폰트가 커진 경우입니다.
페이지는 작은 화면에서 자동으로 검색창/버튼을 더 작게 보여 주도록 설계되어 있습니다.

5) 본문 요약 (간단 / 상세)

폰트 조절 영역 오른쪽의 “본문요약 간단”, “본문요약 상세” 버튼으로 오늘 본문의 흐름을 빠르게 파악할 수 있습니다.

  • 간단 요약: 450–650자, 한 단락으로 핵심 주제·사건 흐름·적용 2–3문장
  • 상세 요약: 3,500–5,000자, 배경·주제·신학적 의미·인물/사건·표 정리(선택)·적용 4–5문장
중요 안내
요약은 구글 AI를 통해 실시간 생성되므로 매번 표현이 조금씩 달라질 수 있습니다.
반드시 “정식 교재”를 먼저 참고하시길 적극 권장드립니다.

7) 구절 설명 (책 버튼)

본문에서 단어나 구절을 드래그로 선택한 뒤, 화면의 책 아이콘(📖) 버튼을 누르면 간단/상세 구절 설명을 볼 수 있습니다.

  • 간단 구절 설명: 350–500자. 지명/인물/성물/상징 등을 간결히.
  • 상세 구절 설명: 2,500–3,500자. 배경·핵심 논지·키워드·성경 간 연결 등 심화 분석.

8) 검색 위치 마커(빨간 점)

같은 단어가 여러 군데에서 발견되면 화면 오른쪽에 빨간 점들이 생깁니다. 각 점은 페이지 내 위치를 나타내며, 점을 누르면 해당 위치로 스크롤됩니다.

(예시) 검색 결과가 여러 건일 때 오른쪽에 빨간 점이 나타납니다. 점을 탭하면 해당 줄로 부드럽게 스크롤됩니다.

9) 중앙 두 줄 강조 스크롤

10) 자주 묻는 질문 & 사용 팁

모바일에서 “남은 시간”이 오른쪽에서 잘려 보입니다.

화면이 매우 좁거나 시스템 폰트 확대가 큰 경우 발생할 수 있습니다. 본 페이지는 작은 화면에서 검색창·버튼·시간 표시를 자동으로 축소하고, 오디오 오른쪽에 약간의 여백을 확보하여 겹침을 완화하도록 설계되어 있습니다.

검색창과 왼쪽 시간(재생된 시간)이 겹칩니다.

모바일에서는 검색창 폭을 뷰포트 기준(vw)으로 자동 축소합니다. 그럼에도 겹치면 가로 모드/세로 모드를 전환하거나, 시스템 글자 크기를 한 단계만 줄여 보세요.

오디오가 재생되지 않아요.

모바일 브라우저는 사용자 입력 없이 자동 재생을 제한합니다. 반드시 재생(▶) 버튼을 한 번 눌러 시작하세요. 네트워크 상태가 불안정하면 잠시 후 다시 시도해 주세요.

키보드/접근성 팁이 있나요?

Tab으로 버튼 이동, Enter로 실행이 가능합니다. 오디오 진행률 막대는 ←/→로 소폭 이동되는 브라우저가 있습니다(브라우저별 상이).

테스트 페이지를 참고하고 싶어요.

제공해 주신 테스트 페이지 예시: HTML-TEST-2.html
상단 검색/오디오 배치, 토글 영역(재생 위치/속도/볼륨), 기도 팝업 등의 동작 흐름은 본 설명서와 동일한 원리로 작동합니다.

도움이 더 필요하신가요?
동작 화면을 캡처하여 문제 상황(기기/브라우저, 세로·가로, 글자 크기 설정)을 함께 알려 주시면 원인 파악에 도움이 됩니다.
